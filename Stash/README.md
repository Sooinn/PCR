# Stash 配置

## 配置说明

此目录包含了优化后的 Stash 配置文件，适用于 iOS 和 macOS 平台。

## 文件列表

- `Stash.yaml`: 主配置文件

## 使用方法

1. 下载 `Stash.yaml`
2. 修改订阅链接
   ```yaml
   proxy-providers:
     机场01:
       url: "替换为你的订阅链接"
   ```
3. 导入到 Stash
